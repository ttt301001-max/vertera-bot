# Vertera Bot

Telegram бот для Vertera — обработка лидов, обучение клиентов и партнёров.

## Деплой на Railway

1. Загрузи эти файлы в новый репозиторий на GitHub
2. Зайди на railway.app → New Project → Deploy from GitHub
3. Выбери репозиторий
4. Перейди в Variables и добавь:
   - BOT_TOKEN = твой токен от BotFather
5. Готово! Бот запустится автоматически.
